**Checkpoint**

- 해시(Hash)
    - ‘키 → 값’을 빠르게 찾기 위한 구조 ⇒ 파이썬의 해시 자료구조: `dict`
- `from collections import Counter`
    - 개수를 세는 특수 딕셔너리
        - 내부적으로 해시 기반 딕셔너리를 사용
    - `Counter(iterable)` : 반복 가능한 객체를 넣으면 자동으로 개수를 세서 `{값: 등장횟수}` 형태로 만들어줌
        - `-` 연산 → 같은 key끼리 개수 빼기
        - 0 이하인 값을 남기지 않음
- `iter(iterable[, sentinel])`
    - 반복 가능한 객체(iterable)을 이터레이터(iterator)로 바꿔주는 함수
        - Iterable : 반복할 수 있는 객체
            - `__iter__()` 메서드를 가지고 있음
            - `for`문에 넣을 수 있음
            - e.g. 리스트, 문자열, 딕셔너리, 튜플
        - Iterator : 값을 하나씩 꺼낼 수 있는 객체
            - `__iter__()` 있음
            - `__next__()` 있음
            - 실제로 값을 꺼내는 객체
    - `iterable`을 계속 호출하다가 결과가 `sentinel`과 같아지면 멈춤
- `next(iterator[, default])`
    - iterator에서 다음 값 하나를 꺼내는 함수
    - 값이 없으면 에러 대신 default 반환
